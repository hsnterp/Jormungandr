# Security policy

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting or open a private security
advisory for this repository. Do not put credentials, exploit details, or other
sensitive information in a public issue.

This project does not require API keys or service credentials. Local datasets,
checkpoints, virtual environments, `.env` files, private keys, and machine-local
agent settings are excluded by `.gitignore`.

## Model safety

Jormungandr is a research model, not a safety-certified earthquake warning or
emergency-response system. See `MODEL_CARD.md` for intended use and limitations.
